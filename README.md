### Swift를 이용하여 간단한 알고리즘 부터 복잡한 알고리즘을 이해하고 학습 하기 위한 공간입니다.

- Sorting Strategy
- Hashing
- DFS & BFS
- Dynamic Programming 
- BackTracking
- Simulation

### Algorithm with Swift의 목적  

- Swift의 전반적인 코딩학습을 위함
- 클로저, 고차함수 등을 적극 활용한 코드 가독성과 간결성 극대화
- C++의 STL과 같은 기능을 Swift에서 구현
